A base class for all series.

Series extends [Component](/api/core/component) and is the foundation for all chart series types.

## Properties

### chart

<ResponseField name="chart" type="Chart | undefined" readonly>
  A chart series belongs to
</ResponseField>

```typescript
const chart = series.chart;
```

### bullets

<ResponseField name="bullets" type="List<(root, series, dataItem) => Bullet | undefined>">
  List of bullets to use for the series
</ResponseField>

See [Bullets](https://www.amcharts.com/docs/v5/concepts/common-elements/bullets/) for more info.

```typescript
series.bullets.push((root, series, dataItem) => {
  return am5.Bullet.new(root, {
    sprite: am5.Circle.new(root, {
      radius: 5,
      fill: series.get("fill")
    })
  });
});
```

### bulletsContainer

<ResponseField name="bulletsContainer" type="Container" readonly>
  A Container series' bullets are stored in
</ResponseField>

  A list of field names to exclude from automatic aggregation when `calculateAggregates` is enabled

## Methods

### startIndex()

Returns current start index of the series.

<ResponseField name="index" type="number">
  Start index
</ResponseField>

```typescript
const startIndex = series.startIndex();
```

### endIndex()

Returns current end index of the series.

<ResponseField name="index" type="number">
  End index
</ResponseField>

```typescript
const endIndex = series.endIndex();
```

### addBullet()

Adds bullet directly to a data item.

<ParamField path="dataItem" type="DataItem" required>
  Target data item
</ParamField>

<ParamField path="bullet" type="Bullet" required>
  Bullet instance (not a function)
</ParamField>

```typescript
const bullet = am5.Bullet.new(root, {
  sprite: am5.Circle.new(root, {
    radius: 5,
    fill: am5.color(0xff0000)
  })
});

series.addBullet(dataItem, bullet);
```

See [Adding directly to data item](https://www.amcharts.com/docs/v5/concepts/common-elements/bullets/#Adding_directly_to_data_item) for more info.

### showDataItem()

Shows series's data item.

<ParamField path="dataItem" type="DataItem" required>
  Data item
</ParamField>

<ParamField path="duration" type="number">
  Animation duration in milliseconds
</ParamField>

<ResponseField name="promise" type="Promise<void>">
  Promise that resolves when animation completes
</ResponseField>

```typescript
await series.showDataItem(dataItem, 500);
```

### hideDataItem()

Hides series's data item.

<ParamField path="dataItem" type="DataItem" required>
  Data item
</ParamField>

<ParamField path="duration" type="number">
  Animation duration in milliseconds
</ParamField>

<ResponseField name="promise" type="Promise<void>">
  Promise that resolves when animation completes
</ResponseField>

```typescript
await series.hideDataItem(dataItem, 500);
```

series.updateLegendMarker(dataItem);

## Heat Rules

Heat rules allow you to set visual properties of series elements based on their values.

### IHeatRule Interface

<ParamField path="target" type="Template<any>" required>
  Target template
</ParamField>

<ParamField path="dataField" type="string" required>
  Which data field to use when determining item's value
</ParamField>

<ParamField path="key" type="string">
  A setting key to set
</ParamField>

<ParamField path="min" type="any">
  The setting value to use for items with the lowest value
</ParamField>

<ParamField path="max" type="any">
  The setting value to use for items with the highest value
</ParamField>

<ParamField path="neutral" type="any">
  The setting value to use for items which do not have value at all
</ParamField>

<ParamField path="minValue" type="number">
  Custom lowest value
</ParamField>

<ParamField path="maxValue" type="number">
  Custom highest value
</ParamField>

<ParamField path="logarithmic" type="boolean" default={false}>
  Use logarithmic scale when calculating intermediate setting values
</ParamField>

<ParamField path="customFunction" type="(target: Sprite, minValue: number, maxValue: number, value?: any) => void">
  A custom function that will set target element's settings
</ParamField>

### Example

```typescript
series.set("heatRules", [{
  target: series.columns.template,
  min: am5.color(0x00ff00),
  max: am5.color(0xff0000),
  dataField: "value",
  key: "fill"
}]);
```

## Data Item Properties

Series data items have the following properties:

<ResponseField name="id" type="string">
  Data item identifier
</ResponseField>

<ResponseField name="value" type="number">
  Main numeric value
</ResponseField>

<ResponseField name="valueWorking" type="number">
  Working value (used during animations)
</ResponseField>

<ResponseField name="valueChange" type="number">
  Change from first value
</ResponseField>

<ResponseField name="valueChangePercent" type="number">
  Percent change from first value
</ResponseField>

<ResponseField name="valueChangeSelection" type="number">
  Change within current selection
</ResponseField>

<ResponseField name="valueChangeSelectionPercent" type="number">
  Percent change within current selection
</ResponseField>

<ResponseField name="valueChangePrevious" type="number">
  Change from previous value
</ResponseField>

<ResponseField name="valueChangePreviousPercent" type="number">
  Percent change from previous value
</ResponseField>

<ResponseField name="customValue" type="number">
  Custom numeric value
</ResponseField>

<ResponseField name="visible" type="boolean" default={true}>
  Whether data item is visible
</ResponseField>

## Examples

### Creating a Series with Bullets

```typescript
const series = chart.series.push(am5xy.LineSeries.new(root, {
  name: "Sales",
  xAxis: xAxis,
  yAxis: yAxis,
  valueYField: "value",
  categoryXField: "category"
}));

series.bullets.push((root, series, dataItem) => {
  return am5.Bullet.new(root, {
    sprite: am5.Circle.new(root, {
      radius: 5,
      fill: series.get("fill")
    })
  });
});

series.data.setAll([
  { category: "A", value: 100 },
  { category: "B", value: 200 },
  { category: "C", value: 150 }
]);
```

### Applying Heat Rules

```typescript
series.set("heatRules", [{
  target: series.columns.template,
  min: am5.color(0x0000ff),
  max: am5.color(0xff0000),
  dataField: "valueY",
  key: "fill"
}]);
```
