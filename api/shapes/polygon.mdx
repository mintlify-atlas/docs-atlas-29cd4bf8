## Overview

The `Polygon` class draws a polygon shape from an array of points. It extends the `Graphics` base class and supports morphing animations between different polygon shapes.

## Class Definition

```typescript
export class Polygon extends Graphics
```

## Settings Interface

```typescript
export interface IPolygonSettings extends IGraphicsSettings
```

## Properties

<ResponseField name="points" type="Array<IPoint>">
  An array of polygon corner coordinates.
  
  Each `IPoint` has the structure:
  ```typescript
  interface IPoint {
    x: number;
    y: number;
  }
  ```
</ResponseField>

<ResponseField name="coordinates" type="Array<Array<number>>">
  Coordinates as an array of arrays.
  
  Each coordinate is `[x, y]` format. This is automatically converted to `points` internally.
</ResponseField>

<ResponseField name="animationDuration" type="number">
  Number of milliseconds to play morph animation.
  
  When the `points` property changes, the polygon will animate from the previous shape to the new shape.
</ResponseField>

<ResponseField name="animationEasing" type="(t: Time) => Time">
  Easing function to use for animations.
  
  [Learn more about easing functions](https://www.amcharts.com/docs/v5/concepts/animations/#Easing_functions)
</ResponseField>

## Example Usage

### Basic Polygon

```typescript
import { Polygon } from "@amcharts/amcharts5";

// Create a triangle using points
let triangle = Polygon.new(root, {
  points: [
    { x: 0, y: 0 },
    { x: 100, y: 0 },
    { x: 50, y: 100 }
  ],
  fill: am5.color(0xff0000),
  stroke: am5.color(0x000000),
  strokeWidth: 2
});
```

### Using Coordinates

```typescript
// Create a polygon using coordinate arrays
let polygon = Polygon.new(root, {
  coordinates: [
    [0, 0],
    [100, 0],
    [100, 100],
    [0, 100]
  ],
  fill: am5.color(0x0000ff)
});
```

### Morphing Animation

```typescript
// Create a polygon that will animate when shape changes
let morphingPolygon = Polygon.new(root, {
  points: [
    { x: 0, y: 0 },
    { x: 100, y: 0 },
    { x: 50, y: 100 }
  ],
  animationDuration: 1000,
  animationEasing: am5.ease.inOut(am5.ease.cubic)
});

// Later, change the shape (will animate)
morphingPolygon.set("points", [
  { x: 0, y: 0 },
  { x: 100, y: 0 },
  { x: 100, y: 100 },
  { x: 0, y: 100 }
]);
```

## Notes

- When morphing between polygons with different numbers of points, the class automatically normalizes both shapes to have the same number of points for smooth animation.
- The polygon automatically calculates its bounding box based on the provided points.
- Available since version 5.4.0
