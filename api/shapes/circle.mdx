## Overview

The `Circle` class draws a circle shape. It extends the `Graphics` base class and provides radius-based rendering.

## Class Definition

```typescript
export class Circle extends Graphics
```

## Settings Interface

```typescript
export interface ICircleSettings extends IGraphicsSettings
```

## Properties

<ResponseField name="radius" type="number">
  Circle radius in pixels.
  
  Default: `10`
</ResponseField>

## Example Usage

```typescript
import { Circle } from "@amcharts/amcharts5";

// Create a circle
let circle = Circle.new(root, {
  radius: 50,
  fill: am5.color(0xff0000),
  stroke: am5.color(0x000000),
  strokeWidth: 2
});
```
